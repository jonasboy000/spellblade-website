Synopsis

Spellblade is a tabletop roleplaying game built to balance depth with ease of use, combining clear rules with enough flexibility for players and GMs to shape the game to their needs. Set in a low-fantasy medieval world, it uses a d100 system to keep outcomes precise and meaningful, with tense, deadly combat designed for use with miniatures. It was created out of the need for a system that didn't fully exist elsewhere—one that offers structure without rigidity, and depth without unnecessary complexity. Drawing inspiration from multiple systems, it introduces mechanics such as a composite dice system for precise damage, a flexible class structure, gear-based progression, and a custom spell creation system that allows players to shape simple but tailored spells, all built to support grounded, consequence-driven play.
